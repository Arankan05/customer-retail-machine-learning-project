# Customer Retail Machine Learning Project

## Project Overview

This project implements and compares multiple Machine Learning classification algorithms using a customer retail dataset. The project demonstrates the complete Machine Learning workflow including data preprocessing, visualization, model training, prediction, evaluation, and performance comparison.

---

# Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

---

# Dataset Features Used

The following dataset features were used in this project:

* Quantity
* UnitPrice
* Country

Other columns available in the dataset:

* InvoiceNo
* StockCode
* Description
* InvoiceDate
* CustomerID

---

# Machine Learning Models Used

## 1. Logistic Regression

Used for binary classification problems.

## 2. Decision Tree Classifier

Tree-based supervised learning algorithm.

## 3. K-Nearest Neighbors (KNN)

Classification based on nearest neighboring data points.

---

# Project Workflow

1. Load customer retail dataset
2. Handle missing values
3. Encode categorical columns
4. Visualize customer data
5. Split dataset into training and testing data
6. Train Logistic Regression model
7. Train Decision Tree model
8. Train KNN model
9. Evaluate models using Accuracy Score
10. Generate Confusion Matrix
11. Compare model performances
